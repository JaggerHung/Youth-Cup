Installation
===
1. Install XAMPP 
2. Setup virtualhost **straitdev.me** for site
2. Browse site at: [http://straitdev.me](http://straitdev.me)

Webpack
===
Open Terminal, go to source folder
```
bower install
npm install
./node_modules/webpack/bin/webpack.js --watch
```


Live Reload
===
Open Terminal, go to source folder
```
npm start
```
